# AgentLair

[AgentLair](https://github.com/piiiico/agentlair-langchain-integration) provides a robust identity layer for LangChain agents. It gives any agent a persistent **@agentlair.dev** email address, an encrypted vault for cross-session state, and an immutable audit trail—all with just a few lines of code.

## Installation

Install the integration package via pip:

```bash
pip install langchain-agentlair
```

## Features

The `langchain-agentlair` package includes five core tools that give your agent a real-world presence:

| Tool | Function |
| :--- | :--- |
| **SendEmailTool** | Send emails from the agent's unique `@agentlair.dev` address. |
| **CheckInboxTool** | Read and process incoming emails. |
| **VaultStoreTool** | Persist secrets or state in an encrypted cross-session vault. |
| **VaultGetTool** | Retrieve stored values from the vault. |
| **LogObservationTool** | Write entries to an immutable audit trail. |

## Setup

To use AgentLair, you'll need an API key and a registered agent email.

```python
from langchain_agentlair import AgentLairClient, agentlair_tools

client = AgentLairClient(api_key="your_api_key")
tools = agentlair_tools(client, "mybot@agentlair.dev")
```

## Usage

### Using with a React Agent

You can use the pre-built `AgentLairAgent` which comes with all tools pre-wired:

```python
from langchain_agentlair import AgentLairAgent
from langchain_openai import ChatOpenAI

agent = AgentLairAgent(
    api_key="your_api_key",
    agent_email="mybot@agentlair.dev",
    llm=ChatOpenAI(model="gpt-4o")
)

agent.run("Check my inbox for any new tasks.")
```

### Callback Tracing

Use the `AgentLairCallbackHandler` to automatically log all agent activity to the immutable audit trail:

```python
from langchain_agentlair import AgentLairCallbackHandler

callback = AgentLairCallbackHandler(client, "mybot@agentlair.dev")
# Pass this callback to your AgentExecutor
```

## Metadata & Persistence

All tools in this package automatically carry `tool.metadata["agent_email"]`. This ensures that downstream tracing and auditing are always tied to the agent's persistent identity, regardless of where the agent is running.
